# MCP Trust Levels

## Level 0: Blocked
No access to MCP tools.

## Level 1: Read-only
Can inspect docs, metrics, and non-sensitive resources.

## Level 2: Limited write
Can update low-risk artifacts with approval gates.

## Level 3: Operational write
Can execute production-adjacent actions with guardrails and logging.

## Level 4: Privileged
Restricted emergency access, short-lived, fully audited.
